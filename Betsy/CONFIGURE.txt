Configuration.

~/.bashrc
~/.genomicoderc
gitpath/Betsy_3version/dot_betsyrc
~/.betsyrc

gitpath is the path where you have your git folder.
You have to change it to your own path in the follow command that have "gitpath".
-------------------------------------------------------
1.Include Betsy to PYTHONPATH

In ~/.bashrc ,add these two lines.

export PYTHONPATH="gitpath"
export PYTHONPATH="gitpath/Betsy_3version:${PYTHONPATH}"

--------------------------------------------------------------

3. In gitpath/Betsy_3version/dot_betsyrc,
set up the outpath for where you want to put your results. The folder of the output path should exist.


--------------------------------------------------------------------
4.symlink the gitpath/Betsy_3version/dot_betsyrc to ~/.betsyrc 
using the command in linux:
ln -s gitpath/Betsy_3version/dot_betsyrc ~/.betsyrc
