Configuration.

~/.bashrc
~/.genomicoderc
gitpath/Betsy/Betsy.cfg
~/.betsyrc

gitpath is the path where you have your git folder.
You have to change it to your own path in the follow command that have "gitpath".
-------------------------------------------------------
1.Include Betsy to PYTHONPATH

In ~/.bashrc ,add these two lines.

export PYTHONPATH="gitpath"
export PYTHONPATH="gitpath/Betsy:${PYTHONPATH}"

--------------------------------------------------------------

3. In gitpath/Betsy/Betsy.cfg, change the gitpath to your own path.
Also set up the outpath for where you want to put your results. The folder of the output path should exist.

The content is:

[Section1]
preprocess = gitpath/scripts/preprocess.py
arrayplot = gitpath/scripts/arrayplot.py
xls2txt = gitpath/scripts/xls2txt
genepattern=gitpath/scripts/run_genepattern.py
bfrmnorm=gitpath/scripts/bfrmnorm.py
combatnorm=gitpath/scripts/combatnorm.py
scoresig=gitpath/scripts/scoresig.py
rename=gitpath/scripts/slice_matrix.py
mapping=/home/changlab/data/mapping/HumanHT-12_to_HG-U133_Plus_2.txt
networkfile=gitpath/Betsy/betsy.xgmml
annotate_matrix=gitpath/scripts/annotate_matrix.py
outputpath= <write your own output path here>

--------------------------------------------------------------------
4.symlink the gitpath/Betsy/Betsy.cfg to ~/.betsyrc 
using the command in linux:
ln gitpath/Betsy/Betsy.cfg ~/.betsyrc
